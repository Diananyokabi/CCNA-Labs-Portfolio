Router


What it does

A router connects different networks together.

Example:

Your home network

The internet

Simple idea
Router = traffic director between networks
Example

When your laptop sends data to a website, the router decides where that data should go on the internet.

2️⃣ Switch

Network switch

What it does

A switch connects devices inside the same network.

Example devices connected to a switch:

computers

printers

servers

Simple idea
Switch = connector for devices in one network

Switches use MAC addresses to send data to the correct device.

3️⃣ Modem

Modem

What it does

A modem connects your home network to your ISP.

It converts signals between:

your ISP’s network

your home network

Simple idea
Modem = door between your house and the internet
4️⃣ Wireless Access Point

Wireless access point

What it does

An access point provides Wi-Fi so wireless devices can connect to the network.

Devices that connect:

phones

laptops

tablets

Simple idea
Access Point = creates Wi-Fi

Many home routers already include an access point.

5️⃣ Firewall

Firewall

What it does

A firewall protects a network from unwanted traffic.

It checks incoming and outgoing data and blocks suspicious activity.

Simple idea
Firewall = security guard for the network
